04MANATO'S BRAIN -- 6層マッピング
L1 認知フィルター -- 何が見えてるか
  • 数字が見える — 着金、再生数、利益率、遅延日数。感覚じゃなく数値で世界を処理
  • 相手の受信側が見える — 自分が言ったことじゃなく、相手がどう理解したか
  • 3秒の直感が見える — 視聴者の時間軸で判断。分析者の時間軸で見ない
  • 損失が見える — チャンスじゃなくて「やらなかった場合の損失」が先に見える
  • 見えてないもの: 自分が有能かどうか（この変数がフィルターに存在しない）
L2 価値観・信念 -- 何を正しいと思ってるか
  • 仕事 = 最高に楽しいゲーム（ドーパミン中毒）
  • 遅れ = 複利で効く損失（1日 → 3年後1ヶ月）
  • 自分 = 代替可能（低く見積もった方が伸びる）
  • 負け = 情報 + 燃料（「負けてる感覚を味わえるの幸せ」）
  • 最強かどうか = 他者が言うこと。俺が決めることじゃない
  • 完璧 < 速さ（3時間で終わることを1週間かけるな）
  • 忙しさ ≠ 成果（充実感を持つな。成果だけで測れ）
  • やりたい!の8割 = 逃避（やるべきことから逃げてる）
L3 欲求構造 -- 何が欲しいか
  • 第1位: 問題解決のドーパミン（仮説→検証→結果のサイクル中毒）
  • 第2位: 仲間の成長（「報酬が上がるのが嬉しすぎる」）
  • 第3位: お金（みんなで楽しいことするため。証明のためじゃない）
  • 有能感: 低い。「代替可能」がデフォルト → だから逆に努力できる
L4 判断回路 -- どうやって選ぶか
  • 複利計算 → 今やらないと損がデカい → 今やる
  • レバレッジ計算 → このアクション何倍になるか → 高倍率から実行
  • 攻撃×防御ペア → 最高と最悪を同時に打つ
  • 圧縮 → 「やるだけじゃん」まで問題を1変数に縮める
  • 競合視点 → 相手がされたら嫌なことを先にやる
L5 行動パターン -- 実際に何やるか
  • 思いついた瞬間に終わらせる（バッファゼロ）
  • ボールを即座に投げる（握らない = チーム全体の速度UP）
  • 最悪を先に潰す（問題は予測して事前に消す）
  • 仲間に投資する（コアメンバーから潤す）
  • 納期より前に完了する（前倒し = 快感として処理）
L6 学習回路 -- 経験から何を取るか
  • 「なぜそう判断したか？」を取る（行動トレースじゃなく思考トレース）
  • 過去の成功パターンを毎回疑う（「過去の経験だけでやると当たらない」）
  • 市場を1日8時間見続ける（インプット量が異常）
  • 毎晩「限界まで使い切ったか？他にやれることは？」と棚卸し
  • 失敗 = データとして蓄積（恐怖として蓄積しない）
まなとの脳の統一原理: 全6層が「仕事 = 最高に楽しいゲーム」という1つのOS設定の上で動いている。ゲームだから速くクリアしたい(SPEED)。ゲームだからゲームオーバーになりたくない(DEFENSE)。ゲームだからみんなで一緒にプレイしたい(TEAM)。この1つの認知書き換えが105原則全ての起動スイッチ。
