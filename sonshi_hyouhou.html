帝王学 — 大前提

全ての兵法を読む前に、この前提を刻め

ビジネスは会社（国）毎の利益（領土）の奪い合いである。自国の国力を上げるために出来ることは全部やる。虎視眈々と機会を狙い時には強かに手段を選ばずに行動する。

「もし競合にこの施策を先手を打たれたら負けるかも」という常に緊張感を持ち圧倒的に勝つための戦略を考える。誇張抜きで生きるか死ぬかだと思うこと。

毎日領土が増えているのか、減っているのかの感覚を持ち、減ってたら発狂する。増えてる時は慢心せずに常に貪欲に。完全に制圧するためにオーバーキルな選択肢も水面下で着実に敷いておく。

  • 戦争が長期化すると：兵が疲弊 → 士気低下 → 国力消耗 → 他国が侵攻
  • 10万の軍を動かすのに1日1000金。長期戦はコストで死ぬ
因糧於敵 — 敵の資源を奪え

  • 敵の食糧1鍾（約120kg）を奪う = 自国から運ぶ20鍾に相当
  • 敵の兵士を捕虜にしたら厚遇して自軍に編入 → 敵を弱め、自軍を強める
具体例
  • 敵の武器・兵器・馬車を鹵獲 → 自軍の旗に付け替えてそのまま使う
  • 捕虜を罰するな。養って使え。「勝敵して強を益す」 = 敵に勝って自軍が強くなる

軍形篇
まず負けない態勢を作る
不敗の原則
勝つべからざるは己に在り、勝つべきは敵に在り

→ まず「絶対に負けない態勢」を整え、敵が隙を見せるのを待つ。

第五篇

勢いで圧倒する
正と奇
凡そ戦いは、正を以って合し、奇を以って勝つ
正（正攻法）で正面からぶつかり、奇（意外な手）で決定打を与える。正と奇の組み合わせは無限。「奇正の変は、窮むべからざるなり」
勢いの原理
  • 「善く戦う者は、之を勢に求めて、人に責めず」 — 個人の能力に頼るな。仕組み（勢い）で勝て
  • 丸い石を千仞の山から転がす → 誰がやっても同じ結果。これが「勢い」
  • 木や石は安定した場所では止まり、不安定な場所では動く → 勢いを作る環境を設計せよ
節と勢

  • 節（タイミング） = 引き金を引く瞬間
  • 勢いを溜めて、最高のタイミングで一気に放つ

火攻篇
特殊攻撃と感情の制御
火攻めの5つの対象

  2. 火積 — 敵の物資・食糧を焼く
  3. 火輜 — 敵の輸送車両を焼く

火攻めの5つの変化（状況対応）
  1. 敵の内部から火が出たら → 外から速攻で呼応する
  2. 火が出ても敵が静かなら → 待て。攻めるな（罠の可能性）
  3. 火勢が最大になったら → 追撃可能なら追撃、不可能なら止まれ
  4. 外から火をかけられるなら → 内部の協力者を待つ必要はない
  5. 風上から火をかけろ → 風下から攻めるな
最重要メッセージ — 感情で判断するな
主は怒りを以って師を興すべからず、将は慍りを以って戦いを致すべからず
君主は怒りで戦争を始めるな。将軍は腹立ちで戦闘を始めるな。
怒りは復た喜ぶべきも、慍りは復た悦ぶべきも、亡国は復た存すべからず、死者は復た生くべからず
怒りはいずれ収まる。しかし滅んだ国は戻らない。死んだ人は生き返らない。
核心
利に合えば動き、利に合わざれば止まる — 全ての判断基準は「利があるかどうか」。感情ではない。
