02核心思想 — 資本主義はゲーム

“勝たないと意味がない。勝ってこそ、
自分たちの考え方が正当化される。
資本主義って、そういうゲームなんで”

GOETHE誌インタビュー原文 — 全4動画の裏側にある共通OS

CORE BELIEF 1
“僕は努力を信仰しない。商売の本質は努力じゃない”

努力信者で商売の本質をつかんでいる人は、あまりいない。タイミング × 夢中が努力量に勝る。

CORE BELIEF 2
“努力は夢中に勝てない”

夢中な人間は寝てる時間以外全部そこにいる。努力で追いつける量じゃない。好きだから熱狂が止まらない状態が最強の燃料。

03資本主義のゲーム — 6つのルール
01
RULE 01 — TIMING × PASSION

勝ち筋は「努力量」ではなく「タイミング × 夢中」

コロナ禍のマスク販売が好例。同じ努力でもタイミングが違えば結果は100倍変わる。「いつやるか」が「どれだけやるか」に勝る。そして待つことも戦略。市場の波を読み取り、正しい瞬間に乗る。

結果 = タイミング × 夢中 >> 努力量
TAIGA CONNECTION

夢中は燃料（加算の力）。Taigaの脱力は到達点（ゼロの状態）。夢中 → 反復 → 回路配線 → 脱力、の順序。同じものではない。

02
RULE 02 — CIRCULATION

お金は「稼ぐもの」ではなく「循環させるもの」

お金目的で動くと、お金が入った瞬間に動機が消える。でもお金を循環の媒体として捉えると、入ってきたお金が次の価値を生み、それがさらに循環する。

お金 = 血液。止まったら死ぬ。循環させて次の価値を生み続けるのが本質。

ゆとりくんが資本金10万で上場できたのは、お金カードが弱くても他のカード（タイミング × 情熱 × SNSの波）で勝てることの証明。

TAIGA CONNECTION

帝王学「Netflix超えて嬉し泣き」がゴールなら、お金は手段。お金を循環させて仲間の才能を開花させる = 従旺格の山（戊）として土台を提供し続ける構造。

03
RULE 03 — SUBTRACTION

やらないことを決める仕組み化（引き算のゲーム）

MEGUMI回の核心。面白い商品は「真面目に考えても生まれない」。ラフに話す中で生まれたアイデアの方が有効。

やることを増やすゲームじゃなくて、やらないことを仕組みで排除して、残った領域に全リソースを注ぐゲーム。MEGUMIの「直感が時代を引き寄せる」理由 = 引き算でノイズを消した結果、本質的な信号だけが見えるようになる。

TAIGA CONNECTION

結果起点I/O（まなと）の引き算版。やることを逆算するだけじゃなく、やらないことを仕組みで先に排除する。龍高星の「面白そう → 飛び込む」衝動に対して「これはやらない」のフィルターを設計で置く。

04
RULE 04 — BIRD'S EYE

「現場が正しい」は逃げ。俯瞰が勝つ。

TBS Bloomberg回の核心。現場主義は心地いいけど、それは目の前の問題を解くだけで全体最適にはならない。

「ビビってる経営者多すぎ」 — 現場に逃げる経営者は設計を放棄してる。俯瞰的視点を持って全体を設計できる人間がゲームを制する。
TAIGA CONNECTION

望遠鏡と顕微鏡の理論と同構造。「宇宙視座で考え目の前を狂気的に実行」= 現場（顕微鏡）に逃げずに俯瞰（望遠鏡）を持つ。

05
RULE 05 — COGNITIVE FILTER

口癖がゲームの入口を決める

「どうせ無理」は認知フィルター。この口癖が走ると、脳が「無理な理由」を自動収集し始める。結果、チャンスの入口が閉じる。

運がいい人の正体 = 気づきの総量が多い人 = フィルターが開いてる人

口癖は無意識の言語化。「どうせ無理」が走ると、認知の入口（felt sense）自体が歪む。運は能力じゃなくて知覚の問題。

TAIGA CONNECTION

言語化の定義と直結。言語化能力が上がる = 自分の口癖 = 無意識の出力を監査できるようになる。felt senseの質が上がれば、気づきの総量が増える。

06
RULE 06 — META GAME

自分が勝つんじゃなくて、勝てる環境を作る

“僕はクリエイターではない。才能を持ってる人がそれを発揮できる環境を作るのが仕事”

「臆病な秀才の最初のきっかけを創る」「ハグレモノをツワモノに」。プレイヤーとして勝つゲームじゃなく、プレイヤーが勝てるフィールドを設計するメタゲーム。

マネジメント禁止、プロデュース重視。作り手に主権を持たせて試行錯誤させる。成長しないブランドは撤退（Yリーグ制度）。

TAIGA CONNECTION

Prestige戦略の完成形。「答えを渡さず問いを渡す」のさらに先。問いすら渡さず、勝てる環境そのものを設計する。ただしTaigaは前線プレイヤー兼設計者のデュアル構造が正解。

04カード交換ゲーム理論
GAME BOARD

資本主義 = カード交換で手札の総価値を上げるゲーム

プレイヤーは手持ちのカード（時間・スキル・人脈・お金・信用・情熱）を交換しながら手札の総価値を上げていく。何を何と交換するかの判断がゲームを決める。努力量じゃなくて判断精度。

05経営哲学 — 補足
熱狂 「好きな人と好きなことに熱狂して働く」
“好きなことを好きな人と一緒にしたいという気持ちが強くて。それが人生で一番幸せなことだと思うんです。でも儲けないと続けられないから、そのために常に戦略的でいます”

熱狂と戦略は矛盾しない。熱狂が方向を決め、戦略がその方向を最短距離で走らせる。
