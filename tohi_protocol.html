ROOT PROBLEM

「逃避か合理か分からない」が根本問題

判断より先に自動反応（3つのブロック発動）が起きる。意志じゃ勝てない。仕組みで勝つしかない。

⚠
DETECTION SIGNS
逃避の兆候（このサインで気づく）
「疲れた」「しんどい」「やる気出ない」
「明日やる」「後でやる」
寝る時間が異常に増える
予定外のタスクに流れる
スマホ・SNS・ポルノ衝動
システム作りで「これ今やる必要？」
「分かりました、やります」で具体ゼロ
考えてばかりで動かない
?
4 QUESTIONS
3秒で出す判別問い
Q1
この行動の結果、何が生まれる？
合理具体的な成果・回復・学び
逃避何も生まれない、時間だけ過ぎる
Q2
意思決定した結果か？
合理「今は休む」と事前に決めた
逃避気づいたらやってた
Q3
向き合うべきタスクから意識が逸れてる？
合理計画的な別タスク・休息
逃避しんどいから回避
Q4
終わった後、後悔する？
合理スッキリ、また向き合える
逃避「時間溶かした...」と思う
2つ以上「逃避側」なら逃避と判定する
3
BLOCK DIAGNOSIS
どのブロックが動いてる？
完璧主義
完璧にやらなきゃ → できない → 逃げる
攻撃偏重
向き合う（守り）が苦手 → 攻めやすい別タスクに流れる
個人主義
一人で抱え込む → しんどい → 逃げる
共通の根：有能感の防衛
向き合う → 失敗するかも → 有能感傷つく → 予防のために回避（無意識・瞬時）
4
COUNTERMEASURES
対策4層
LAYER 01 — 事前設計
プリコミットメント（逃避する前に決める）
時間割固定／スマホ別室／サイトブロッカー／物理的に逃げ場を消す
LAYER 02 — 自動化
If-Then プラン（意志力を使わない）
「もし○○と感じたら → △△をする」を事前に決めておく。下のIf-Then集を参照
LAYER 03 — リアルタイム検知
手書きトリガー × 10分タイマー
手に「逃避×」を書く／10分おきに「この10分、結果に繋がってた？」を自問
LAYER 04 — リカバリ
逃避してしまった時の戻り方
自己否定NG（攻撃偏重発動 → 次の逃避を生む）／事実だけ記録／構造分析／次の1分だけ決める
→
IF-THEN PLANS
事前に決めておく自動反応
もし「向き合うの辛い」と感じたら
→
5分だけ始める（5分ルール）
もし「疲れた」と感じたら
→
20分タイマーで散歩（PFCリセット）
もし「スマホ見たい」と感じたら
→
スマホを別室に置く、10秒待つ
もし「眠い」と感じたら
→
立つ・水を飲む・光を浴びる
もし「完璧にやらなきゃ」と感じたら
→
60点で出すと決めて始める
もし「明日やる」と思ったら
→
今5分だけ手をつける（ゼイガルニック効果）

RECOVERY
逃避してしまった時の3ステップ
STEP 01
事実だけを記録（感情抜き）
「○分間、○○をしてた」だけ書く。「俺はダメだ」は禁止
STEP 02
構造を見る
3つのブロックのどれが動いてた？ どの判別問いに該当した？
STEP 03
次の1分だけ決める
1時間じゃなく1分。ハードルを徹底的に下げる
