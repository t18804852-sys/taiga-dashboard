マインドセット（前提）
1
常にYouTubeを見続ける。1日8時間競合の動画やアナリティクスを見続けるのが大前提。
2
常にWhyを深堀り続け自分の頭で考える。浅い分析では伸びない。なぜ？を何回も深ぼると人の感情にたどり着く。
3
徹底的な視聴者目線を持つ。ペルソナと同じ人を1人用意して地道に聞いていくのが大正解。
4
YouTubeの目的 = 通りすがりの人をファンにすること = 視聴者の感情を揺らすこと。
5
サムネ → タイトル → 冒頭 → 台本の順で設計。入口から設計し、中身を後から合わせる。
6
数字で判断し感覚を疑う。感覚で仮説 → 数字で検証 → 言語化して蓄積。
7
ベースのクオリティ x 継続。ホームランより毎打席ヒット。
1
リサーチ（企画・サムネタイトル）
🔍 企画リサーチ手順
  1. 再生回数 > チャンネル登録者の動画を10個ほどリサーチ（投稿日時が最近であればあるほど良い）
  2. 競合チャンネルで伸びてる動画と伸びてない動画の違いを「企画サムネタイトル」と「中身」の2軸で分析
  3. なぜ伸びているかを最小単位の複数の軸で分析（企画・タイトル・サムネ文言・台本構成・内容・演者ビジュアル・抑揚・リズム・背景・音質・画質・画角・編集テンポ）
  4. コメント欄やコメントへのいいねから視聴者が求めていたことを分析
  5. 伸びる要素を組み込んだサムネ案・タイトル案を決める（必ずサムネ → タイトル → 台本構成 → 内容の順）
📊 市場調査
  • キーワード検索ボリューム調査（Googleトレンド / YouTube検索サジェスト / VidIQ）
  • 競合TOP5〜10の登録者・平均再生数・投稿頻度を一覧化
  • マーケットの天井見積もり（特化ジャンルTOPの登録者数 = 天井の目安）
  • トレンドの方向性判断（縮小トレンド → 実力を20%上げないと現状維持すら不可）
💡 企画の原則
YouTubeにオリジナリティは不要。答えはYouTube市場にある。「抽象 ⇔ 具体」で企画を捻出する。
企画の判断基準：「サムネが想像できるか」
👤 ペルソナ作成
  • 層 / 名前 / 性別 / 年齢 / 家族構成 / 仕事 / 年収
  • 人生の目標 / ジャンルへの知識 / 悩み / 価値観 / 休日の過ごし方
✅ チェックリスト
  • 再生回数 > 登録者の競合動画を10本リサーチした
  • 伸びてる/伸びてない動画の差を2軸で分析した
  • コメント欄から視聴者ニーズを抽出した
  • キーワード検索ボリュームを調査した
  • 競合TOP5〜10を一覧化した
  • ペルソナを作成した
  • サムネが想像できる企画に落とし込んだ
2
サムネ・タイトル制作
🎯 サムネ制作6ステップ
  1. 目的定義 — この動画で何を達成するか
  2. 観察 — 競合サムネを大量に見る
  3. 分析 — 伸びてる理由を要素分解
  4. 設計 — 4軸チェックで構成
  5. 具体化 — 実制作
  6. 検証 — ABテストで数字確認
📏 4軸チェック
軸役割例
インパクト目が止まる大きな数字、強い色、驚きの表情
具体何の話か理解「在宅ワーク」「物販」等
ベネフィット自分に得がありそう「月収60万」「時給3万円」
好奇心もっと知りたいモザイク、矛盾ワード、意外な組み合わせ
2024年までは3軸で勝てたが2026年は好奇心がないと差別化できない
✍️ 好奇心コピーの型
  • 矛盾型 — 常識と逆のことを言う
  • 数字ギャップ型 — 意外な数字で驚かせる
  • 否定型 — 「〇〇するな」「〇〇は嘘」
  • 暴露型 — 裏側・真実を匂わせる
  • ボタンの掛け違い型 — ズレた組み合わせ
🔨 制作ルール
  • Z構図で左上に最強要素を配置
  • サムネ x タイトル x 冒頭 = 3点セットで設計
  • 2〜3パターン用意 → ABテスト
  • サムネとタイトルの文言は意図的にずらす
  • サムネは明るい方が良い（シンプルに目立つ）
📈 影響度順位
コピー > 表情 > 配色 > フォント > レイアウト
サムネのROIが最も高い（改善だけで再生数1.5倍の事例あり）
⏱️ サムネ差替え判断基準
15分後CTR判断
6%以下秒で差替え
6〜9%30分まで様子見
9%以上キープ
✅ チェックリスト
  • 競合サムネを20枚以上観察した
  • 4軸（インパクト/具体/ベネフィット/好奇心）を全て満たした
  • 好奇心コピーの型を適用した
  • Z構図で左上に最強要素を配置した
  • サムネとタイトルの文言をずらした
  • サムネ x タイトル x 冒頭の3点セットが連動している
  • 2〜3パターンのサムネを用意した
  • 文字は大きく、要素は少なく、明るさを確保した
  • スマホサイズでの視認性を確認した
  • コピーが最も影響度が高い要素として練り込んだ
  • 表情に感情が乗っている（素材が使える場合）
  • 配色は3色以内に絞った
  • YouTube上で並んだ時に埋もれないか確認した
  • ペルソナが思わずクリックする内容になっている
3
台本制作
🔥 ディレクター必読マインド
1. 添削依頼時の台本提出は200%で作りこめている状態で提出。制作をKPIに置くマインドは捨てる。使命はアベレージPVの上昇・売上の増加のみ。
2. 全ての文章に意図を持って書く。意図のない言葉は単語単位で入れない。意図を聞かれた時に即答できない状態は論外。
📝 台本制作フロー
  1. 参考動画の洗い出し・リサーチ
  2. 抽出した要素を書き出す
  3. 構成の作成
  4. 執筆
🎯 台本作成ポイント
  1. 3つほどの異なる競合動画の構成と内容の共通項を書く。バズる要因の内容はしっかり抑える。自社の優位的要素（権威・社会的証明・実績・オリジナルノウハウ・演者の感情）を盛り込む
  2. 演者が8割。演者の人柄・喋り方・考え方・夢を深く理解した上で「その演者が言いそうなこと」を書く
  3. 機能的価値か感情的価値の要素が入っているか確認
  4. 書き言葉ではなく話し言葉で書く。完成台本を最低3回音読
  5. 接続詞を使いこなす（視聴者が「全く予測不可能」を防ぐ）
📐 構成ルール
  • 最初と最後に衝撃的な情報を配置
  • 冒頭30秒で複数ペルソナを列挙 + 2つの見どころ予告
  • 前振りは最長1分30秒以内
  • 宣伝セリフは冒頭NG（画像LINE誘導はOK）
🔗 回遊施策
  • 概要欄にリンク
  • 編集でサムネ挿入
  • 白テキストでタイトル記載
✅ チェックリスト
  • 3つ以上の競合動画の構成・内容の共通項を書き出した
  • 自社の優位的要素（権威/実績/ノウハウ/感情）を盛り込んだ
  • 演者の人柄・喋り方に合った言葉で書いた
  • 機能的価値 or 感情的価値が含まれている
  • 話し言葉で書き、最低3回音読した
  • 冒頭30秒で複数ペルソナ列挙 + 見どころ予告を入れた
  • 前振りは1分30秒以内に収まっている
  • 最初と最後に衝撃的な情報を配置した
  • 冒頭に宣伝セリフが入っていない
  • 全ての文章に意図がある（即答できる状態）
  • 回遊施策（概要欄リンク/サムネ挿入/白テキ）を仕込んだ
  • 200%の完成度で提出できる状態
4
撮影
🎬 演者の心構え
  1. 本来の自分の演者力を最大限引き出す。人を真似た言葉遣い・ジェスチャー・抑揚はNG
  2. 「視聴者にこうなってほしい」というピュアな感情を持って撮影。打算的はダメ
  3. ロボットにならない。文字が想起させるイメージを思い浮かべながら喋る
  4. 綺麗な言葉で淡々と喋るより多少汚い言葉でも熱量がこもっている方が3倍良い
  5. 緊張は論外。視聴者に伝えたいのは自分の熱意
  6. 100人中1人だけに刺しにいくマインドで話す
😊 気持ちよく撮影してもらうコツ
  • 恥ずかしさを捨てる
  • 台本に面白いを散りばめる
  • 普段使わない言葉を極限まで削る
📷 撮影環境・設定
  • WB / ISO / 明るさ全オート、露出 -0.7
  • 有線マイク推奨
  • 1回で4〜6本撮影
  • 日中の自然光推奨。背景は観葉植物やPCモニターなど
  • HD 30fps / HDRオフ / 16:9
✅ チェックリスト
  • 台本を十分にリハーサルした
  • カメラ設定（露出-0.7 / HD 30fps / HDRオフ / 16:9）を確認した
  • 有線マイクを接続した
  • 日中の自然光を確保した
  • 背景の見え方を確認した
  • 演者がリラックスした状態で撮影開始できた
  • 100人中1人に刺すマインドで収録した
  • 熱量が乗った状態で撮り切った
5
編集
🎬 編集の基本
  • 冒頭不要パートカット（本題に最速で入る）
  • LINE誘導: 冒頭（10秒以内）+ 途中 + 最後で画像付き
🎵 BGM
  • 音数が少ない曲を選ぶ（音量を下げるのではない）
  • シーンの切り替えでBGMも切り替え
💬 テロップ
  • 先出し厳禁（テロップが先に出ると視聴者が読んで離脱）
  • フォント・大きさ・エフェクトをシーンで使い分け
  • 感情に合わせたデザイン
💥 SE・フリー素材
SE / フリー素材の挿入は最低5秒に1回
📱 ショート動画
  • ロングの切り抜きが基本
  • 冒頭2秒勝負
✅ チェックリスト
  • 冒頭の不要パートをカットした
  • LINE誘導を冒頭10秒以内 + 途中 + 最後に配置した
  • BGMは音数が少ない曲を選んだ
  • シーン切り替えでBGMも切り替えた
  • テロップの先出しがない
  • テロップのフォント/大きさ/エフェクトをシーンで使い分けた
  • SE/フリー素材を最低5秒に1回挿入した
  • ショート用の切り抜きを選定した
  • 全体のテンポ感を確認した
6
投稿・分析・改善
📤 投稿ルール
  • ロング週1〜2本 + ショート
  • ショートの活用: 投稿期間が空く時の繋ぎ + 通常尺動画への誘導（視聴回数の0.6〜1.6%がロングへ）
  • 投稿直後にサムネABテスト: 必ず複数枚用意してCTR確認
🔍 VSEO（動画SEO）
  • 市場が大きいジャンルや初期は検索流入を狙う
  • VidIQでAve Views 10万以上のワードを狙う
  • 初期戦略: 流入元と流入先の属性を合わせる
🤖 アルゴリズム評価軸
CTR > 視聴継続率 > 平均視聴時間
📊 分析方法
  • トラフィックソース分解
  • グルーピング機能
  • 相対的視聴維持率（PC版のみ）
  • 月次振り返り: 月間再生数アベレージを前月比で上げ続ける
❤️ ファン化・長期戦略
  • 演者のプライベートや人間性を出す。ノウハウの話に絡めて1〜2文程度で散りばめる
  • 脳内SEO: コミュニティ投稿・LINE・メルマガで「忘れられないための施策」
  • 長期再生型: 教科書となるような何度も見返す動画が最も売上を作る
✅ チェックリスト
  • サムネを複数枚用意してABテストの準備をした
  • 投稿15分後にCTRを確認・差替え判断した
  • VSEO（タイトル/説明/タグ）を最適化した
  • トラフィックソースを分解して分析した
  • 相対的視聴維持率をPC版で確認した
  • ショート動画で繋ぎ + ロングへの誘導を設計した
  • コミュニティ投稿/LINE等で脳内SEO施策を実施した
  • 月間再生数アベレージを前月比で確認した
